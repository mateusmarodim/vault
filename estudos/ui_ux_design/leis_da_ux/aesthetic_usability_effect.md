# Efeito Estética-Usabilidade

Os usuários tendem a ser mais tolerantes a problemas de usabilidade leves quando a interface é esteticamente agradável.

Um design esteticamente agradável cria uma resposta positiva no cérebro, e leva as pessoas a acreditarem que o design efetivamente funciona melhor. 

Designs visualmente agradáveis podem mascarar problemas de usabilidade e prevenir a descoberta de defeitos durante testes de usabilidade.
